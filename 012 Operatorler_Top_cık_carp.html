JavaScript Atama Operatörleri Atama operatörleri, JavaScript değişkenlerine değerler atar. Operator Örnek Neye Eşit = x = y x = y += x += y x = x + y -= x -= y x = x - y *= x *= y x = x * y /= x /= y x = x / y %= x %= y x = x % y **= x **= y x = x ** y Operatör Tanım Açıklama = Eşittir Değişkene değer / veri atamak için kullanılır. Çoklu atamalarda gerçekleştirebilir. += Toplama ataması Değişkene değer atanırken, değişkenin kendi değeri, diğer değişkenin değeri ile toplanarak çıkan sonuç değişkene atanmış olur -= Çıkartma ataması Değişkene değer atanırken, değişkenin kendi değerinden, diğer değişkenin değeri çıkartılarak çıkan sonuç değişkene atanmış olur. *= Çarpma ataması Değişkene değer atanırken, değişkenin kendi değeri, diğer değişkenin değeri ile çarpılarak çıkan sonuç değişkene atanmış olur. /= Bölme ataması Değişkene değer atanırken, değişkenin kendi değeri, diğer değişkenin değerine bölünerek çıkan sonuç değişkene atanmış olur. %= Modülüs ataması Değişkene değer atanırken, değişkenin kendi değeri, diğer değişkenin değerine bölünmesi sonucunda kalan değer değişkene atanmış olur. **- Üs al ataması Değişkene değer atanırken, değişkenin kendi değeri, diğer değişkenin değeri ile üssü işlemine tabi tutularak çıkan sonuç değişkene atanmış olur. JavaScript Kaçırma Operatörleri bazı değişkenleri tanımlarken kullanılan kaçış operatörleridir sorunsuz atama yapımasını sağlarlar \b (backspace) (geri al) \f (form feed) (form besleme) \n (new line) (yeni satır) \r (carriage return) (paragraf sonu) \t (tab) (yatay sekme) \v (vertical tab) (dikey sekme) \' (tek tırnak) \" (çift tırmak) \\ (ters slash)
